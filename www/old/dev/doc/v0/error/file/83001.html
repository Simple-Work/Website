Erreur non traitée

L'erreur 83001 arrive l'orsqu'un script capte une erreur (exemple : dans un try catch) mais ne la traite pas. Alors il renvoie le message de l'erreur original + le code 83001 .
En général, cela ce produit dans les fonctions encore en test qui comporte un try catch uniquement pour des raison de sécurité.
